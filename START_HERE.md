# 🎯 START HERE - AI Translator

## ⚡ Super Quick Start

### Birinchi marta ishga tushirish:

```bash
npm run install:all
npm run dev
```

### Keyingi safar:

```bash
npm run dev
```

## 🌐 Open in Browser

```
http://localhost:5173
```

## 📝 Barcha Buyruqlar

| Buyruq | Vazifasi |
|--------|----------|
| `npm run dev` | Backend va Frontend ni ishga tushirish |
| `npm start` | `npm run dev` bilan bir xil |
| `npm run install:all` | Barcha paketlarni o'rnatish |
| `npm run dev:backend` | Faqat Backend |
| `npm run dev:frontend` | Faqat Frontend |
